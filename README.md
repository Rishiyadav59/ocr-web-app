# AI OCR Web System

A lightweight full-stack application that extracts text from images using Tesseract.js.

### 🛠️ Tech Stack
- **Backend:** Node.js, Express
- **OCR Engine:** Tesseract.js
- **Frontend:** Vanilla JS & HTML (Server-rendered)

### 🚀 How to Run
1. `npm install`
2. `node server.js`
3. Open `http://localhost:5000`
